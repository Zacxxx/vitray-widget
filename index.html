Glassmorphic desktop widget for system monitoring and terminal access.

Installation

Add the repository and install:

# Add repository
echo "deb [trusted=yes] https://$(echo $GITHUB_REPOSITORY | cut -d'/' -f1).github.io/$(echo $GITHUB_REPOSITORY | cut -d'/' -f2) stable main" | sudo tee /etc/apt/sources.list.d/vitray-widget.list

# Update and install
sudo apt update
sudo apt install vitray-widget

Updates

Updates will be automatically available via:

sudo apt update && sudo apt upgrade

Manual Installation

Download the latest .deb package from the releases page.
